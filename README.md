# Smart Student Performance Prediction System

## 📌 Project Overview
The Smart Student Performance Prediction System is a Machine Learning project that predicts student academic performance based on various input features. It helps identify students who may require additional academic support by analyzing educational data.

## 🎯 Objective
- Predict student performance using Machine Learning.
- Analyze the factors affecting academic performance.
- Assist educators in making data-driven decisions.

## 🛠️ Technologies Used
- Python
- Google Colab
- Jupyter Notebook (.ipynb)
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Dataset
The dataset contains student-related information such as:
- Study Hours
- Attendance
- Previous Scores
- Assignments
- Extra Activities
- Final Performance

## ⚙️ Machine Learning Workflow
1. Import Dataset
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Performance Prediction
